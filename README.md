# Simple Python Calculator

A functional Command Line Interface (CLI) calculator built using Python 3. This project focuses on clean logic, user input validation, and graceful error handling.

## Overview
This calculator allows users to perform basic mathematical operations while ensuring the program does not crash due to invalid inputs or illegal mathematical operations (such as dividing by zero).

## Features
* **Basic Arithmetic:** Supports standard operations: `+`, `-`, `*`, `/`, and `%`.
* **Input Validation:** Uses `try-except` blocks to ensure only integers are processed, preventing crashes from non-numeric input.
* **Error Handling:** * Detects and alerts the user for unsupported operators.
    * Prevents ZeroDivisionError by checking logic before calculation.

## Requirements
* **Python Version:** Python 3.6 or higher.
* **External Libraries:** None (Uses only the Python Standard Library).
